## 🌟 Overview

**Academic ERP System** is a powerful and user-friendly web application designed for colleges, departments, and academic institutions to manage daily administrative operations efficiently.

From student records to attendance tracking, this platform transforms manual work into a fast, secure, and intelligent digital workflow.

---

## ✨ Key Features

✅ Secure Login & Registration System
✅ Student Data Management (Add / Edit / Delete / Search)
✅ Attendance Entry & Tracking
✅ Dashboard with Real-Time Statistics
✅ Export Records to Excel
✅ Session-Based Authentication
✅ Clean and Responsive UI
✅ Easy to Customize for Any Institution

## 🚀 Installation & Run

```bash
git clone https://github.com/Darpan2000-web/academic-erp-system.git
cd academic-erp-system
python main.py
```

Then open:

```text
http://127.0.0.1:5000
```
